Qué hace esta app

  • Permite preparar publicaciones para TikTok (principalmente como borradores) desde un flujo automatizado.
  • Se usa para contenido propio de Fantasy Dugout MLB (resúmenes, imágenes, clips, etc.).

Cómo funciona

  1. El usuario autoriza TikTok (OAuth).
  2. La app envía el video/imagen a TikTok para crear un borrador o post según permisos.
  3. El usuario revisa y publica manualmente (si aplica).

Nota: el comportamiento exacto depende del estado de revisión/auditoría y permisos habilitados.

Datos y permisos

Usamos los permisos estrictamente necesarios para publicar contenido. No vendemos datos. Más detalles en Privacy Policy.
