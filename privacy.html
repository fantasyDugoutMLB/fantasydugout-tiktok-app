Privacy Policy

Última actualización:

1) Quiénes somos

Fantasy Dugout (“nosotros”) opera esta aplicación para automatizar la creación de publicaciones/borradores en TikTok para contenido propio.

2) Datos que recopilamos

  • Tokens de acceso (OAuth) necesarios para autenticar con TikTok.
  • Identificadores técnicos (p. ej. IDs de publicación) para verificar estados de carga.
  • Contenido subido (videos/imágenes) que tú proporcionas para publicar.

3) Para qué usamos los datos

  • Autenticar tu cuenta y ejecutar acciones de publicación/borrador.
  • Monitorear el estado de procesamiento y mostrar resultados.
  • Soporte y resolución de incidencias.

4) Qué NO hacemos

  • No vendemos tu información.
  • No usamos tus datos para publicidad comportamental.

5) Compartición de datos

Compartimos datos únicamente con TikTok para ejecutar la funcionalidad solicitada por ti y con proveedores técnicos estrictamente necesarios (hosting/almacenamiento), si aplica.

6) Retención

Conservamos tokens e identificadores el tiempo necesario para operar el servicio o hasta que solicites su eliminación/revoques el acceso.

7) Seguridad

Aplicamos medidas razonables para proteger credenciales y datos (p. ej. cifrado en tránsito, control de accesos).

8) Tus derechos

Puedes solicitar acceso/eliminación de datos y revocar permisos desde TikTok. Ver Data Deletion.
